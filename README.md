## Sobre mim:
  Faço iniciação científica no laboratório de Bioinformática, em conjunto com o laboratório de Protozoologia da Universidade Federal de Santa Catarina (UFSC). No laboratório, trabalho com o processamento e análise de dados provenientes de infecções, principalmente relacionadas a tripanossomatídeos. ## About me:
I am an undergraduate researcher in Bioinformatics Laboratory, in collaboration with the Protozoology Laboratory at the Federal University of Santa Catarina (UFSC). In the lab, I work with the processing and analysis of data derived from infections, mainly related to trypanosomatids. My activities include data organization, computational analysis, and the use of programming languages such as Python and Shell Bash, which has strengthened my interest in programming and data analysis applied to real biological problems.
